    Installing MSYS2

    Step 1 of 4

    MSYS2 provide a unix terminal environment for Windows. We will need this to run the build commands to create our SplashKit programs.

    1. Download the correct version of MSYS (32bit or 64bit depending on your computer).

    2. Run the installer and follow the installation wizard, with default values. (Do not install in a location with spaces in the path!)

    3. Follow the installation steps on the MSYS2 page by executing the command pacman -Syu.

    4. Restart your terminal and relaunch MSYS2, then execute pacman -su

    Once you have completed these step, move on to the next step - installing SplashKit.
